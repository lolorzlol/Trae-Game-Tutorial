# 测试驱动开发 (TDD) 案例

## 概述

使用 Superpowers 的 `test-driven-development` skill 进行测试驱动开发。

基于 Unity Test Framework 框架。

## 安装 Unity Test Framework

1. 打开 `Window > Package Manager`
2. 安装以下两个包（可能已自带）：

![alt text](image-9.png)

## 对话流程

```
严格使用 superpowers 工作流，我要实现 [功能描述]，请使用 TDD 方式开发
```

之后的头脑风暴流程参考 [直接创建Demo项目](Tutorial-1/Examples/直接创建Demo项目.md)

## 关键点

TDD 采用 **Red-Green-Refactor** 迭代开发：

| 阶段 | 说明 |
|------|------|
| Red | 先写失败的测试 |
| Green | 开发代码使测试通过 |
| Refactor | 重构代码 |
